# bioC_onnector
Bioelectricity connector

# Dependency

* boost::asio::beast
* boost::property_tree

# Build

* install boost v1.68

```sh
$ sudo add-apt-repository ppa:mhier/libboost-latest
$ sudo apt-get update
```
* run `build.sh`
